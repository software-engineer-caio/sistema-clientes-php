# Sistema de Gerenciamento de Clientes

Este é um sistema web simples para gerenciamento de clientes.

## Funcionalidades

- Cadastrar clientes
- Listar clientes
- Buscar clientes por nome
- Atualizar dados de clientes
- Excluir clientes

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Integração

O sistema utiliza a API ViaCEP para preencher automaticamente o endereço a partir do CEP.
